## Introduction

I am using vim and nvim as my primary text editors. I also heard about emacs which i might try in future. for now this is my vim configuration guide for my personal Reference.

So basically vim is improved version of vi editor written in C language, nvim is improved version of vim.

> Why i use vim/nvim ?

- Lightweight (No additional browers running in background)
- Fast (No switching between mouse and keyboard)
- Highly customizable
- Works in terminal
- Large community and plugins ecosystem
- Available everywhere ( almost all unix based systems have vim preinstalled )
